# 🧠 Parkinson’s Disease Detection

A Flask web app that predicts Parkinson’s disease using a trained SVM (Support Vector Machine) model.  
It takes 22 biomedical voice parameters as input and provides an instant prediction result.

---

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/Sushant-Dagar/Parkinsons-Disease-Detection.git
cd Parkinsons-Disease-Detection

2. Install required packages

pip install -r requirements.txt

3. Run the Flask app

python app.py

4. Open in browser

Go to:

http://127.0.0.1:5000
